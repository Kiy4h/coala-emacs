Flycheck coala Checker
======================

Integrate [coala](https://coala.io) with
[flycheck](http://www.flycheck.org).

Setup
-----

Put this in your Emacs initialization script.

```elisp
(eval-after-load 'flycheck
  '(require 'flycheck-coala))
```
